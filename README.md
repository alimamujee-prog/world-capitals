# World Capitals

A cinematic study tool and showcase for all **195 sovereign world capitals** — the 193 UN member states plus Vatican City and Palestine. Live at **[www.worldcapitals.co](https://www.worldcapitals.co)**.

Browse, flip flashcards, quiz yourself, and hear pronunciations. Built as a single self-contained `index.html` (no build step).

## How to make changes

1. Edit `index.html`.
2. Save, then run:
   ```
   git add -A && git commit -m "describe your change" && git push
   ```
3. Vercel auto-deploys the new version to the live site within a minute.

## What's here

- `index.html` — the entire site (data, design, and logic in one file)
- `vercel.json` — hosting config
- `world-capitals-flashcards.html` — the original baseline this was rebuilt from (not served)
